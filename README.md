# Python Quiz Game 🧠

A simple quiz application built in Python that asks True/False questions and tracks your score.

## 📌 Features

- Loads questions from external data
- Uses object-oriented programming
- Tracks score in real time
- Simple command-line interface

---

## 🚀 How It Works

1. Questions are loaded from `data.py`
2. Each question is turned into a `Question` object
3. `QuizBrain` controls quiz flow
4. User answers True/False
5. Score updates automatically
